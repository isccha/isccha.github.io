Chatbot

Developed from scratch a chatbot for the Universidad de Costa Rica using the Telegram API. The chatbot offers menus with information of the Electrical Engineering career and it’s capable of understanding general questions. Information sent includes text, attached documents, pictures and videos.

At the same time, statistics and useful information is stored in a database (MySQL). An administrator is capable of sending custom messages to audience (groups or private chats) on behalf of chatbot’s name, and a webpage for monitoring stats were also created using SQL Data Extraction. Some of the stadistics are: number of messages in the last week/month, most asked question, most asked topic, questions the chatbot were not able to understand.

For this project the main code was written in PHP and different scripts in Perl were created for administrator activities (messages, stats monitoring). Git SSH keys were configured in the server so the administrator can change the chat contents externally.
